# 📸 Evidence

Evidence is a mobile application that allows users to capture photos and mint them as NFTs on the blockchain, creating a permanent and tamper-resistant record.

The app is designed to provide a **verifiable timestamped record** using blockchain technology.

---

## ✨ Features

- 📷 Capture photos as evidence
- 🪙 Mint photos as NFTs
- ⛓ Immutable blockchain record
- 🕒 Timestamp verification via blockchain
- 🔐 User-controlled ownership

---

## 🧩 How It Works

1. Capture a photo using the app  
2. The image is prepared for minting  
3. Mint the image as an NFT on the blockchain  
4. The NFT acts as a permanent, timestamped record  

---

## ⚠️ Important Disclaimer

- This app does NOT provide legal certification of evidence  
- NFTs do NOT guarantee legal admissibility in court  
- Users are responsible for how they use the app  

---

## 🔗 Blockchain

- Network: Solana (or compatible)
- NFTs are minted directly by users
- Ownership is controlled by the user wallet
